# Task Manager App

A RESTful API to manage your daily tasks using Node.js, Express, and MongoDB.

## Features

 Add new tasks
- Get all tasks
- Update or delete a task
- MongoDB Atlas connected
- Mongoose schema with validation

## Run the Project
bash
npm install
node app.js
